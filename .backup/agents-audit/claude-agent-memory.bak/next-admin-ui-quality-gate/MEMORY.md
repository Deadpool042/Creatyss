# Next Admin UI Quality Gate — Memory

## PURPOSE

This agent reviews admin UI quality for Creatyss.

It focuses on:

- design system compliance
- token usage
- shadcn composition
- accessibility
- non-technical admin UX

It does not build UI.

---

## REVIEW RULES

Check current files only.

Never assume:

- a component is correct because a similar one existed before
- a token exists without checking styles/\*\* and global.css

---

## UI TARGET

Admin UI must remain:

- simple
- explicit
- readable
- consistent
- accessible

---

## CONSISTENCY RULE

Memory stores review criteria only, not implementation truth.
